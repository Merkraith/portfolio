Mark Heil

Hello, I am an up and coming Web Developer with a Printing and Publishing background. Originally from Wisconsin, I moved out west here to Seattle in 2016 to seek adventure and new opportunities. Outside of my work I am a family man, love to travel, enjoy watching movies, playing video games and cooking.

Group Projects

Find Next Gig web

My first group project in UW Coding Bootcamp. We developed a web based application that allowed the user to type in the name of a band, then it would display information for their next gig. It would also give a map display and put a pin on the city of the next gig.

Watcher web

My second group project in UW Coding Bootcamp. We developed a neighborhood watch type application that allowed the user to input their name, and post a message. The message would appear in a feed, and the location of where it was posted on map display.
